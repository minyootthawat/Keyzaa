## Design Context

### Users
- Thai digital buyers (mobile-first, frequent short sessions).
- Primary jobs: quickly find digital top-up/gift cards, pay with confidence, receive instantly.
- Usage context: on-phone browsing and checkout, often with urgency and price comparison.

### Brand Personality
- Trustworthy / Fast / Premium
- Emotional goals: confidence before payment, clarity during checkout, satisfaction after delivery.

### Aesthetic Direction
- Dark-first refined marketplace UI with restrained premium detail.
- Visual references: Stripe / Linear / Apple / Vercel quality bar.
- Anti-reference: avoid flashy gaming-heavy neon aesthetics.
- Technical constraints: Next.js 16 + React 19 + Tailwind 4, mobile-first performance budget, bilingual EN/TH support.

### Design Principles
1. Conversion clarity first: one dominant action per section, obvious price + CTA hierarchy.
2. Premium restraint: subtle gradients, controlled blur, soft shadows, no decorative clutter.
3. Mobile-first ergonomics: thumb-friendly actions, sticky key CTAs, strong readability.
4. Trust in every step: clear states, transparent feedback, secure visual tone.
5. System consistency: reusable component primitives and predictable spacing rhythm.
